# HyperFrames

使用 HTML、CSS 和 JavaScript 创建 AI 视频，通过 Hermes Web UI 渲染为 MP4。适合短片头、电影预告片、产品宣传、字幕动画、HUD/科技视觉等场景。

## 功能

- 用 Web 技术（HTML/CSS/JS）定义视频帧
- 支持关键帧动画、CSS transition、JS 驱动的动态效果
- 自动渲染为 MP4 视频
- 支持自定义分辨率、帧率、时长

## 前置条件

- Hermes Agent 已安装并运行
- Hermes Web UI 已启动（含 HyperFrames 渲染能力）

## 使用方式

直接描述你想要的视频效果：

- "做一个 5 秒的科技感片头，标题是 HyperFrames"
- "创建一个产品发布的动画，有渐入和缩放效果"
- "做一个 HUD 风格的数据可视化动画"

Hermes 会自动生成 HTML/CSS/JS 代码并通过 Web UI 渲染。

## 适用场景

- 短视频片头/片尾
- 产品宣传动画
- 数据可视化动画
- 字幕/标题动画
- 科技感 HUD 风格视觉
- 网页转视频
